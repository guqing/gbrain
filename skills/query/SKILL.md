# Query Skill

Search and retrieve knowledge from the brain.

## Commands

- `gbrain search <query>` — Keyword (FTS5) search
- `gbrain query <query>` — Semantic vector search (requires OPENAI_API_KEY)
- `gbrain get <slug>` — Get a page by slug
- `gbrain list` — List all pages

## Options

- `--type <type>` — Filter by page type
- `--limit <n>` — Max results
- `--json` — Output as JSON
