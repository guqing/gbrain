# Ingest Skill

Import and harvest data into the brain.

## Commands

- `gbrain import <file>` — Import a single markdown file
- `gbrain sync <dir>` — Sync a directory of markdown files
- `gbrain harvest` — Harvest from external sources (Phase 3)

## Options

- `--no-embed` — Skip embedding generation during import/sync
- `--json` — Output results as JSON
