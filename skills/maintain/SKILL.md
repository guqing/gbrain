# Maintain Skill

Keep the brain healthy and up to date.

## Commands

- `gbrain health` — Show health metrics (coverage, staleness, orphans)
- `gbrain lint` — Detailed lint report
- `gbrain versions <slug>` — Manage page version snapshots
- `gbrain embed` — Generate embeddings for pages
- `gbrain stats` — Brain statistics

## Options

- `--json` — Output as JSON
- `--snap` — Create a snapshot (versions command)
- `--revert <id>` — Revert to a version (versions command)
